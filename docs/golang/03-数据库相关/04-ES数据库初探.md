# ElasticSearch
- 基于索引的搜索方式，文档存储数据
- 存储方式
    - index索引
        - 类型 type(相当于数据库中的表)
            - 文档-在索引的类型下就可以添加文档了
                - 字段
                    - 数据（类型属性等的描述）
- json格式的查询语言
- restful API 风格的接口
- 使用 get、put、post、delete很方便修改文档

## RESTful 设计风格简述
- GET 获取
- POST 修改
- PUT 添加
- DELETE 删除
